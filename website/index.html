🌙

NonSleep

Keep your Mac awake with the lid closed.

$ brew install nonsleep
NonSleep: ON
◎

Menu Bar

One-click toggle from your menu bar. Nothing more.

⌨

CLI

Full terminal control. nonsleep, nonsleep stop, done.

☽

Lid-Aware

Detects lid close. Sleeps display, keeps system alive.

⚙

Native

Pure Swift + IOKit. Near-zero memory. No Electron.

Usage

# Enable sleep prevention
nonsleep

# Disable
nonsleep stop

# Check status
nonsleep status

# Toggle
nonsleep toggle

# Temporary — 2 hours
nonsleep for 2h

Build from Source

git clone https://github.com/3289david/nonsleep.git
cd nonsleep
make install
